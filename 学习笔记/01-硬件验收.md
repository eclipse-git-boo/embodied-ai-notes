# 硬件验收与安全清单

每项通过后再勾选；照片、串口日志和标定文件放入相应实验目录。不要让模型直接以满速控制未标定机械臂。

## 通用

- [ ] 记录电脑 OS、NVIDIA 驱动、CUDA、Python 版本和 `nvidia-smi` 输出。
- [ ] 机械臂安装在刚性桌面；工作区无松动物和线缆干涉。
- [ ] 有可立即断电的急停/电源开关，操作者清楚位置。
- [ ] 首次控制采用低速、低扭矩、单关节，验证正负方向。
- [ ] 设置软件关节限位和工作空间限位；验证限位触发时行为。

## SO-ARM101

- [ ] 确认 follower 和 leader 分别可被系统识别，记录端口与设备 ID。
- [ ] 执行官方标定；保存标定结果、版本与日期。
- [ ] 手动遥操作 5 分钟无丢帧、突跳或过热；记录控制频率。
- [ ] 相机固定，测得图像分辨率、FPS、曝光；保存一张工作区参考图。
- [ ] 标出相机、机械臂基座和任务区位置；不要在采集途中移动它们。

## 松灵 NERO（先确认型号）

- [ ] 写明完整型号、控制器、自由度、末端执行器和厂商文档链接。
- [ ] 确认可用控制接口（ROS/ROS 2、SDK、CAN、串口或 Ethernet）与状态反馈。
- [ ] 获取 URDF/关节限制/坐标系定义；单关节低速验证方向。
- [ ] 若为移动平台，额外验证遥控、制动、避障和失联停车。

# 具身智能：从机械臂到 VLA / WAM

本仓库用于学习一条可重复的具身开发闭环：**硬件标定 → 遥操作采集 → 数据质检 → 模仿学习训练 → 真机评测与部署 → VLA / WAM 研究复现**。

现有资源：RTX 4090 48 GB、SO-ARM101 一套、松灵 NERO 设备一套。当前资料目录为空，本仓库从路线、实验协议和记录模板开始建设。

## 当前模式

当前电脑仅用于学习与资料整理：不安装训练环境、不连接硬件、不执行部署。所有命令和真机步骤均是供未来训练/真机机器使用的参考，不应在此电脑直接执行。见 [学习模式说明](学习笔记/03-学习模式说明.md)。

## 学习时先理解什么

第一个里程碑不是直接微调大 VLA，而是在 SO-ARM101 上完成一个可量化的单任务闭环（推荐：`抓取桌面色块并放入指定盒子`）：

1. 读懂一次机械臂任务从观测到动作的完整数据流；
2. 以 ACT 为参照理解“非 VLA”策略的训练与失败来源；
3. 对比 SmolVLA、OpenVLA 与 π₀ 的动作表示和迁移路径；
4. 理解 WAM 如何以动作条件预测辅助动作选择；
5. 将硬件和数据采集步骤记录为未来可执行的实验设计，而非现在部署。

详细执行次序见 [路线图](学习笔记/00-总路线图.md)，每次实验使用 [实验记录模板](学习笔记/模板-实验记录.md)。
当前阅读可从 [六周学习计划](学习笔记/04-六周学习计划.md) 开始，并用 [从论文到工程的知识地图](学习笔记/05-从论文到工程的知识地图.md) 检查自己是否只理解了模型、却遗漏了数据与闭环。
网页发布配置见 [GitHub Pages 发布说明](学习笔记/07-GitHub-Pages发布.md)。

## 目录

- `VLA/`：VLA 论文导读、开源复现决策与笔记。
- `WAM/`：World Action Model（WAM）概念、论文导读和最小实验。
- `学习笔记/`：路线、硬件、数据、训练、部署与实验日志。

## 关键原则

- 先让小模型在真实硬件上稳定成功，再追求“大模型/泛化”。
- 数据与评测协议和模型权重同等重要；每个结论必须能回溯到数据版本、代码提交和视频。
- 第一次真机部署限速、设硬件急停和软限位，任何异常立即停止。
